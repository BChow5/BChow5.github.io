    This project was for a web based application that helped children learn more English vocabluary through fun and interactive games. It was designed for parents and teachers who want more engaging ways for their children to improve vocabulary. It motivated children by incorporating game-like elements such as scoreboards, timed game modes, and achievement milestones.

    Features

    • Vocabulary and definition quizzes.
    • Scoreboards to track progress and compete with others.
    • Timed game mode to challenge users.
    • Achievement milestones to celebrate and track progress.
    • Customizable difficulty levels.

    Technology Used

    • Frontend: HTML, CSS, JavaScript
    • Backend: Python, Flask
    • Database: SQLite
    • API Integration: Dicionary API

    Future Improvements

    • Allow users to create custom avatars.
    • Provide analytics for children's strengths, weaknesses, and areas of improvement.
